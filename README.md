## 🚀 How to Deploy (Free) with GitHub Pages

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** button → **New repository**
2. Name it `focusflow`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. Click **uploading an existing file**
2. Drag ALL files from this folder into the upload area
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to **Settings** → **Pages**
2. Under "Source" select **Deploy from a branch**
3. Choose **main** branch, **/ (root)** folder
4. Click **Save**

### Step 5 — Your app is live!
After ~2 minutes, your app will be at:
```
https://YOUR-USERNAME.github.io/focusflow/
```

## 📱 How Students Install It

### Android (Chrome)
1. Open the link in Chrome
2. A banner appears: **"Install FocusFlow"** — tap it!
3. Or tap the ⋮ menu → **"Add to Home screen"**
4. It appears on your home screen like a real app ✅

### iPhone / iPad (Safari)
1. Open the link in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add** ✅

### Desktop (Chrome / Edge)
1. Open the link
2. Click the install icon in the address bar (looks like a monitor with ↓)
3. Click **Install** ✅
